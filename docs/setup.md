# Setup Guide

## Prerequisites
- Node.js (v16 or higher)
- npm or yarn

## Installation
1. Clone the repository
2. Run `npm install`
3. Copy `.env.example` to `.env` and configure
4. Run `npm run dev` to start development server

## Directory Structure
- `src/` - Source code
- `public/` - Static assets
- `tests/` - Test files
- `dist/` - Production build output
